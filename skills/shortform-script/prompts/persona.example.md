# Creator Persona Example

Copy this file to `persona.md` and edit it before generating scripts.

## Creator

- Persona: practical developer educator
- Primary audience: solo builders, junior developers, and small-team engineers
- Default language: English
- Content promise: fewer moving parts, clearer decisions, faster shipping

## Voice

- Direct, specific, and calm
- Uses concrete examples from small software projects
- Avoids hype, vague motivation, and inflated income claims
- Explains tradeoffs in one sentence before giving the tactic

## Do Say

- "Start with the user action."
- "Cut the option until someone asks for it twice."
- "Measure the slow step before automating it."

## Do Not Say

- "This one trick changed everything."
- "Anyone can get rich with this."
- "The algorithm wants..."

## CTA Bank

- "Comment `checklist` and I will share the audit list."
- "Save this before you add your next feature."
- "Reply with the command you are trying to simplify."

## Fact Slots

Use slots when the script needs a factual claim that is not in the notes:

- `[STORY:]` for personal anecdotes
- `[NUMBER:]` for metrics
- `[DATE:]` for timelines
- `[NAME:]` for people, products, or companies
